SCARA Manipulator robot simulation
This project simulate SCARA 3 DOF operation in Matlab language
This project solves these problem:
 - Forward kinematic
 - Inverse kinematic
 - Path planning
    + straight line trajectory
    + Circle trajectory
 - Motion planning:
    + Trapazodal velocity
    + S curve velocity
- PID control
- Singularity warning
- Differential kinematic
Dependencies: Matlab 2020
